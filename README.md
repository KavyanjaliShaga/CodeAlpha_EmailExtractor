# Email Extractor

This is a Python-based Email Extractor project developed as part of the CodeAlpha Internship.

## Features
- Extracts email addresses from text files
- Stores extracted emails in an output file

## Files Included
- email_extractor.py – Python script for extracting emails
- input.txt – Input text containing emails
- emails.txt – Output file with extracted emails

## How to Run
1. Make sure Python is installed
2. Run the file using:
   python email_extractor.py
